# StarBox Backend (Node.js)

Este é o backend refeito em Node.js/Express para o projeto StarBox.

## Como rodar

1. Instale as dependências:
   ```bash
   cd backend-js
   npm install
   ```
2. Inicie o servidor em modo desenvolvimento:
   ```bash
   npm run dev
   ```
   Ou em modo produção:
   ```bash
   npm start
   ```
3. O backend estará disponível em http://localhost:3001

## Endpoints iniciais
- `GET /health` — Health check
- `GET /api/hello` — Exemplo de endpoint

Adicione novas rotas conforme a necessidade do frontend.
